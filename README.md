# AuthDevtool

This project is based on NextJS

## Requirements

This project requires the following to get started:
- Node.js version `22.20` or higher

## Getting Started

Clone the repository
```bash
git clone https://github.com/RIFRAICH/auth-devtool.git
cd manager/
```

### Setup
1. Copy `.env.example` and rename it to `.env.local`
2. Update the values in `.env.local` to match your environment

Install dependencies
```bash
npm install
```

Run the project
```bash
npm run dev
```

## Development

Format code with Prettier
```bash
npm run format
```
